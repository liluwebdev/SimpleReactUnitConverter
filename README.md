# React Unit Converter 🔢

A simple **unit converter** built with **React**. Supports conversions for:
- **Meters ↔ Feet**
- **Kilograms ↔ Pounds**
- **Celsius ↔ Fahrenheit**
- **Feet ↔ Meters**
- **Pounds ↔ Kilograms**
- **Fahrenheit ↔ Celsius**

## Features
Real-time conversion  
Responsive design  
React Hooks (`useState`)
